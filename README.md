# Mix Game

플러터로 개발된 카드 게임 웹 애플리케이션입니다.

## 프로젝트 소개

이 프로젝트는 플러터 웹을 사용하여 개발된 카드 게임입니다. 사용자들은 다양한 카드 게임을 즐길 수 있습니다.

## 주요 기능

- 카드 게임 플레이
- 반응형 웹 디자인
- 모던한 UI/UX

## 기술 스택

- Flutter
- Dart
- Web

## 시작하기

### 필수 조건

- Flutter SDK
- Dart SDK
- 웹 브라우저

### 설치 방법

1. 저장소 클론
```bash
git clone https://github.com/snosnosno/mixgame.git
```

2. 의존성 설치
```bash
flutter pub get
```

3. 실행
```bash
flutter run -d chrome
```
